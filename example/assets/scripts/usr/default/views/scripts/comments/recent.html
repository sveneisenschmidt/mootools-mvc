Recent Comments

This List of comments is retrieved from assets/scripts/usr/default/data/comments.json
through a Model called Comments (Default_Comments_Model).

It gets rendered by a view script called comments/recent.html.

There are only 3 entries but you can set a lower amount of results returned
by changing/adding the last url paramter like:

  • #/comments/recent/1 (Get only one entry)
  • #/comments/recent/50 (Get fifty entries)
